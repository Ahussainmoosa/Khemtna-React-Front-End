## User Stories
_________________________________________________________
•	As a user, I want to browse available camps.
•	As a user, I want to have an option of filtering the search
•	As a user, I want a search bar to search the camps by their names. 
•	As a user, I want to see the photos and locations for the camps.
•	As a user, I want to leave a review after I have booked the camps.
•	Asa user, I want to leave a review after staying.
•	As a user, I want to book a camp for a specific date.
•	As a user, I want to view the history of my books.
## Admin Stories
•	As an admin, I want to create owner accounts. 
•	As an admin, I want to be able to delete user accounts.
•	As an admin, I want to delete owners.
## Owner Stories
•	As an owner, I want to add my camps name , photo, price ,description , els.
•	As an owner, I want to edit my camps details, photo, price ,description , els.
•	As an owner, I want to have a setting section that  I can edit my personal details.
•	As an owner, I want to see the booking history and the book dates.
•	As an owner, I want to view my income history .
